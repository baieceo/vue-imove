# vue-imove

## Project setup
```
yarn install
```

### Install imove-cli
```
yarn global add @imove/cli
```

### Run imove-cli
```
imove -d
```

### Compiles and hot-reloads for development
```
yarn serve
```

### Open Browser
```
http://localhost:8080/
```

### Customize configuration
See [Configuration Reference](https://cli.vuejs.org/config/).
